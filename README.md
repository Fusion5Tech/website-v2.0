# Next.js frontent version of our organization's website.

## Cloning the repo

```bash
git clone https://github.com/Fusion5Tech/website-v2.0.git
```

## Cganging Directory
```bash
cd website-v2.1
```

## Installig Node Modules

#### Using pnpm
```bash
pmpn install
```

#### Using nnpm
```bash
mpn install
```
#### Using yarn
```bash
yarn install
```

## Runnign the project

### Using pnpm
```bash
pnpm dev
```

### Using npm
```bash
npm dev
```
### Using yarn
```bash
yarn dev
```

## Opening the project

 Open your browser and got to [localhost:3000](http://localhost:3000)
